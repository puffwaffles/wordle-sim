Instructions

  1. You have 6 tries to guess a random 5 letter word
  2. You can only use existing words. If the word does not exist, you will be prompted to enter the word again without using a try
  3. Guessed letters will be colored based on location in the mystery word.
    • Gray means the letter is not in the word. Excess duplicates of a letter will also be marked as gray if they are in the wrong position.
    • Orange means the letter is in the wrong position.
    • Green denotes that the letter is in the right position
  4. If you get stuck, you can always use the helper tool to help find words to enter.
    • Green rules provide words with a matching letter at a given position (index starting at 0).
    • Orange rules provide words with a matching letter that is not at a given position (index starting at 0).
    • Gray rules provide words that have the same amount of occurrences for a given letter and frequency.
    • The priority rule filters out words that can not be used as an answer.
    • The unique rule provides all words with at least as many unique letters as the given amount.
    • Individual rules can be removed by pressing the x on the rule's tab. All rules can be removed by pressing the reset button

Playing with Agent Mode

  1. In this game mode, the agent will be playing alongside the player.
  2. The agent will only make its move after the player enters a valid word
  3. The player will not see the letters the agent use. Also the agent has no knowledge of what the player chooses.
  4. The player wins only if they find the winning word on a turn before the agent finds the word.
  5. If the player finds the word on the same turn as the agent or if both run out of turns, the player and agent will be tied.
  6. Otherwise, the agent wins if it finds the word first.

Agent Only Mode

  1. In this game mode, the agent will be the one deciding which words to use.
  2. To make the agent choose a word, press the enter button.
